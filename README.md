# Amigo Secreto 🎁

Este es un pequeño proyecto web que permite ingresar nombres de amigos y sortear un "Amigo Secreto" al azar.

## 🚀 Cómo ejecutar el proyecto

1. **Clona este repositorio** o descarga los archivos en tu computadora.
    ```sh
    git clone https://github.com/Carlos-Leguizamo/ChallengeAmigoSecreto.git
    ```

2. **Abre el archivo `index.html` en tu navegador web**.
    - Debes de tener instalado Live Server como extensión de VSCode.
    - Dentro del archivo, presiona clic derecho y selecciona "Open with Live Server".
    - Luego de ello, ya tienes el juego en tu interfaz del navegador.

## 📂 Estructura del Proyecto
```
AmigoSecreto/
│── index.html       # Estructura principal de la página
│── style.css        # Estilos de la página
│── app.js           # Lógica del sorteo y validaciones
└── assets/          # Imágenes y otros recursos
```

## 🎮 Uso
1. Escribe los nombres en el campo de texto y haz clic en "Añadir".
2. Una vez agregados al menos dos nombres, haz clic en "Sortear amigo".
3. Se mostrará el nombre seleccionado y se limpiará la lista.

¡Diviértete con tu sorteo de Amigo Secreto! 🎉
